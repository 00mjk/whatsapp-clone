<h3 align="center">WhatsApp Clone</h3>

## Introduction:

This application is a whatsapp clone using React + firebase

## Start:

```sh
$ git clone git@github.com:caioagiani/whatsapp-clone.git
```

```sh
$ cd whatsapp-clone
```

```sh
$ yarn install
```

```
$ yarn start
```
